# Store Sales Performance Dashboard

## Project Description
A comprehensive multi-page Power BI retail sales performance dashboard tracking year-over-year performance, district-level analytics, chain comparisons, and new store performance across multiple locations.

## Dashboard Preview
![Store Sales Dashboard](Store Sales.pdf)

## Key Features
- Multi-page sales performance analysis
- Year-over-year sales comparison
- District and chain-level performance tracking
- Sales variance analysis by month and category
- New store performance monitoring
- Interactive fiscal month filtering

## Data Sources
- Store sales transaction data
- Chain and district organization data
- Fiscal calendar information
- District manager assignments
- New store launch data

## Tools Used
- Power BI Desktop
- DAX for variance calculations
- Time intelligence functions
- Hierarchical data modeling
- Geographic mapping capabilities

## How to View
1. Download the .pbix file from this repository
2. Open with Power BI Desktop
3. Refresh data connections to load latest sales data
4. Use filters to analyze specific time periods or districts
