# Power BI Projects Portfolio

## Overview
This repository showcases my Power BI dashboards focused on business intelligence, analytics automation, and interactive reporting. Each project demonstrates real-world use cases, from data modeling to DAX calculations and report navigation.

## Repository Structure
Each project folder contains:
- Exported PDF or static screenshots for quick viewing
- Project insights and findings summary
- Data source details
- Supporting files such as Excel or CSV (if applicable)

## Skills Demonstrated
- Power Query transformations (M Language)
- Data modeling and relationship management
- DAX measures and calculated columns
- Bookmarks, drill-through, tooltips, and report navigation
- KPI tracking and business metrics reporting
- Performance optimization for large datasets

## How To Use
1. Follow the link provided in each Project Readme
2. Open it in Power BI Desktop (latest version recommended)
3. Explore filters, pages, bookmarks, and cross-highlights
