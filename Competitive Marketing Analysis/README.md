# Competitive Marketing Analysis Dashboard

## Project Description
A comprehensive Power BI marketing performance dashboard analyzing sales channels, ROI, and product performance across multiple regions and time periods.

## Dashboard Preview
![Competitive Marketing Dashboard](Competitive Marketing Analysis.pdf)
Web Preview: https://app.powerbi.com/groups/me/reports/9d26bf41-4fca-458e-8858-b7b98c84fcc8/ReportSectione0912ffb2ba909ea50d0?experience=power-bi

## Key Features
- Multi-page marketing performance analysis
- ROI tracking across different sales channels
- Geographic sales distribution by state
- Product-level profitability analysis
- Time-series trend analysis
- Channel performance comparisons

## Data Sources
- Multi-channel sales data (Online, Social Media, Stores, Outlet)
- Regional sales data by state
- Product performance metrics
- ROI calculation data

## Tools Used
- Power BI Desktop
- DAX for ROI calculations
- Time intelligence functions
- Geographic mapping (Azure Maps)

## How to View
1. Download the .pbix file from this repository
2. Open with Power BI Desktop
3. Enable Azure Maps visual if needed for geographic analysis
